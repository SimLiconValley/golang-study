# Go Study Repository

이 저장소는 [A Tour of Go](https://go.dev/tour/welcome/1)를 통해 Go 언어의 기초를 다지는 학습 공간입니다.

## 학습 목표

- Go 언어의 기본 문법 및 흐름 제어 이해
- 데이터 구조(Struct, Slice, Map) 및 타입 시스템 숙달
- 메서드와 인터페이스를 통한 객체지향적 설계 이해
- Go의 핵심인 동시성(Concurrency) 모델 학습

## 학습 자료

- [A Tour of Go (공식 사이트)](https://go.dev/tour/welcome/1)
- [Go Documentation](https://go.dev/doc/)

## 진행 상황 확인

학습 진도는 [A Tour of Go 학습 기록](https://github.com/SimLiconValley/golang-study/issues/1) 이슈에서 관리합니다.

## 디렉토리 구조

- `/docs`: 학습 관련 문서 및 PlantUML 다이어그램
- `/src`: (예정) 각 챕터별 예제 코드 및 연습 문제 풀이
